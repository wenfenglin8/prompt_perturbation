# RQ2 150 项扩展分析

本报告在合并原始 n50 GPT/main 数据和 add100 GPT/main 数据后，重新运行 RQ2 的漂移-性能分析。该设计包含 factual QA、math reasoning 和 code generation 三类任务，每个任务 150 个 item；每个 item 有 5 次 original generation，并在 5 类 perturbation 条件下各有 5 次 generation。

最终 item-perturbation 表包含 n = 2250 个观测（三类任务 x 150 个 item x 5 类 perturbation）。语义漂移使用扣除噪声后的 Sentence-BERT 余弦距离衡量。性能变化定义为 original performance 减去 perturbed performance，因此正值表示性能下降。

## 整体漂移-性能关联

合并全部 2250 个观测后，Spearman rho = 0.105（p = <0.001；Holm 校正后 p = <0.001）。Pearson r = 0.167（p = <0.001）。与较小规模的 RQ2 分析类似，Pearson 和 Spearman 结果存在差异，说明这种关联更容易受到分布形态和高杠杆单元的影响，而不是在所有 perturbation 和所有任务上都表现为稳定的单调关系。

## Factual QA 稳健性检查

对于 factual QA，合并所有 perturbation 后的原始 Spearman 关联为 rho = 0.234（p = <0.001）。这是与早期 n=50 factual QA 结果最直接对应的扩展版本，但现在使用的是 150 个 factual QA item x 5 类 perturbation = 750 个观测。

加入 perturbation fixed effects 后，漂移系数 beta = 0.674，naive OLS p = <0.001。使用 item-clustered standard errors，并以 G = 150 个 item cluster 下的 t(G-1) 作为参考分布时，beta = 0.674，SE = 0.291，t = 2.314，p = 0.022。渐近正态 cluster p-value 为 0.021。与早期只有 10 个 cluster 的稳健性检查相比，扩展后的样本拥有足够多的 cluster，因此 cluster-robust approximation 不再那么脆弱。

在 perturbation 内部进行中心化后，Spearman rho = 0.213（naive p = <0.001），这描述了扣除 perturbation-type 均值后的剩余连续关联。

## 对 RQ2 的回答

扩展后的每任务 150 个 item 分析，比早期每任务 10 个 item 的 RQ2 运行提供了更稳定的估计。所有任务合并后的 Spearman 关联仍然较弱，因此不应把 RQ2 回答为“语义漂移和性能下降之间存在简单的全局单调关系”。最强证据仍然是任务条件性的：factual QA 显示出正向的 drift-performance 关联，而且更大的 item 数量使得控制 perturbation、按 item 聚类的稳健性检查比原始小样本分析更可信。Math reasoning 和 code generation 应根据各自任务行进行解释，而不应从 pooled trend 直接推断。

Paraphrasing 仍然是最值得检查的主要 perturbation，因为它通常比其他 perturbation type 产生更大的语义漂移和更大的平均性能变化；但本报告保留了多重比较校正后的 p 值。最稳妥的结论因此是条件性的，而不是普遍性的：语义漂移对 factual QA 最具解释性，而跨任务 pooling 会掩盖任务和 perturbation family 之间的异质性。
